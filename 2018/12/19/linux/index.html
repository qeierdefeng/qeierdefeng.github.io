文件管理命令练习

  1. 在终端当前目录下建立子目录a1,a2，a3。
    mkdir a1 a2 a3
  2. 在目录a1下利用touch命令建立文件f1.txt，并利用echo命令向f1.txt中输入文本，并显示f1.txt的文本内容。（文本内容随意）
    cd a1
    touch f1.txt
    echo hello world > f1.txt
    cat f1.txt
  3. 利用cat命令建立文件f2.txt，并向f2.txt中输入文本，并显示f2.txt的文本内容。(文本内容随意)
    cat >> f2.txt<<EOF
    cat f2.txt
  4. 把f1.txt和f2.txt合并成f3.txt。
    cat f1.txt f2.txt>f3.txt
  5. 把f1.txt移动到目录a2中。
    Mv f1.txt ../a2
  6. 把f2.txt改名为ff.txt,并复制到a2中。
    Cp f2.txt ../a2/ff.txt
  7. 把a2中的f1.txt文件压缩，再解压缩。
    tar -czvf f1.txt.tar.gz f1.txt
    tar -xzvf f1.txt.tar.gz
  8. 把a2中的两个文件打包为指定文件名的文件再拆包。
    tar -cvf f.txt.tar f1.txt ff.txt
    tar -xvf f.txt.tar
  9. 把a2中的两个文件打包并压缩为指定文件名的文件再解压拆包。
    tar -czvf f1.txt.tar.gz f1.txt ff.txt
    tar -xzvf f1.txt.tar.gz
  10. 删除非空子目录a2,删除空目录a3。
    rm -r a2
    rmdir a3
  11. 观察新建文件以
    及新建文件夹的访问权限，查看本机的权限掩码，修改文件f3.txt的访问权限为所有用户均可读可写可执行
    ll
    umask
    chmod 777 f3.txt
  12. 寻找ls命令的二进制文件及其帮助文件。
    Whereis ls
  13. 查看ls命令的帮助手册。
    Man ls
  14. 比较man sleep和man 3 sleep结果的差别。
    Man sleep 查看使用命令
    man 3 sleep查看库函数

  15. 运用cat命令建立文本文件f1.txt和f2.txt,输入不低于10行的英文文本。（运用输出重定向符和输入重定向结束符）
    cat >>f1.txt<<EOF
    EOF

    cat >>f2.txt<<EOF
    EOF

  16. 在当前用户的宿主目录下搜索所有以txt为扩展名的文件并输出重定向到文本文件temp中，再显示文件temp的内容。
    find /home/liangliang -type f -name “*.txt” >>temp.txt
    cat temp.txt

  17. 在当前用户的宿主目录下搜索所有以txt为扩展名的文件并显示查找到的文件的内容
    （比较4、5两条命令结果）
    find /home/liangliang/ -type f -name “*.txt”| cat
  18. 搜索根目录下的所有普通文件并分屏显示。
    find / -type f | more
  19. 查找所有与abc相关的文件并分屏显示。
    find /home/liangliang/ “abc” |more
  20. 在当前用户宿主目录下搜索所有权限为775的文件
    find /home/liangliang/ -perm 775
  21. 在当前用户宿主目录下搜索该用户的所有文件并分屏显示
    find /home/liangliang/ *|more
  22. 统计文件f1.txt的行数、字数、字符数。
    wc f1.txt
  23. 查找文件f1.txt中某一指定文本，如“abc”所在的行以及行号。
    Grep -n “list”f1.txt
  24. 对文件f1.txt的各行进行排序。
    Sort -n f1.txt
    1. 查找文件f1.txt中所有以字母”e”结尾的单词。
      Grep “e>”f1.txt
  25. 统计当前目录下所有普通文件数
    find ./ -type f |wc -l

    用户管理命令练习

  26. 显示/etc/passwd的文件内容，查看当前登录用户的信息
    cat /etc/passwd
  27. 显示/etc/group的文件内容，查看组信息
    cat /etc/passwd
  28. 使用useradd创建用户user1，并自动创建用户主目录，指定uesr1属于当前登录用户所属的组
    sudo useradd -m user1 -g liangliang
  29. 使用adduser创建用户user2
    sudo adduser user2
  30. 为user1和user2设置口令
    sudo passwd user1
    sudo passwd user2
  31. 再查看/etc/passwd和/etc/group的文件内容
    cat /etc/passwd
    cat /etc/group
  32. 修改user1所属组为user2
    usermod -g user2
  33. 分别切换至用户user1和user2，在两个用户的家目录下新建文件夹aa
    su user1
    cd /home/user1
    mkdir aa
  34. 切换至超级用户root，并在sudo的配置文件/etc/sudoers中给user1和user2授权
    su
    vi /etc/sudoers
    user1 ALL=(ALL:ALL) ALL
  35. 删除用户user1和user2
    userdel -r user1 user2
  36. 登录系统，新增一个名为some的组。
    Sudo groupadd some
  37. 新增一个名为erdi的用户，指定erdi的所属组为some，指定其主目录为/home/erdi，指定其shell为/bin/bash。
    Sudo useradd erdi -g some -d /home/erdi -s bin/bash
  38. 添加一个用户名为student组名为students,初始密码为123456。
    Sudo adduser student
    sudo groupadd students
    sudo gpasswd -a student students
  39. 向some组中添加用户tom。
    Sudo useradd tom
    sudo gpasswd -a tom some
  40. 切换到用户erdi（切换时切换用户工作环境），在其家目录目录下建立文本文件f1.txt、f2.txt。
    Su
    cd~
    sudo chmod u+w /etc/sudoers
    vim /etc/sudoers
    chmod u-w /etc/sudoers
    su erdi
    cd /home
  41. 切换到root用户，并修改erdi的密码。
    Su
    passwd erdi
  42. 更改文件f1.txt的所属用户为tom。
    chown tom f1.txt
  43. 更改文件f2.txt的所属组为students。
    chgrp -R students f2.txt
  44. 删除组some。
    groupdel some

    磁盘管理命令练习

  45. 查看当前系统硬盘情况以及具体信息。
    df
  46. 创建硬盘分区，对一个硬盘创建三个分区：主分区、扩展分区以及逻辑分区。熟练掌握创建硬盘分区的新建、删除、改变分区类型号码等命令
    sudo fdisk /dev/adb 进行分区
    m 菜单
    p 查看硬盘分区情况
    n 创建新的分区
  47. 为各分区创建文件系统
    mkfs -t ext3 /dev/sdb1 创建文件系统
  48. 挂在磁盘分区
    sudo mkdir /mnt/sdb1
    suod mkdir /mnt/sdb2
    sudo mount -t ext3 /dev/adb1 /mnt/sdb1
    sudo mount -t ext3 /dev/adb2 /mnt/sdb2
  49. 查看挂载情况
    df
  50. 卸载磁盘分区
    sudo umount /dev/sdb1
